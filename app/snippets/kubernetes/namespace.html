Create a Namesapce in K8s using YAML

Published by Tim Roberts on April 5th, 2022 at 7:10 PM | Updated on April 5th, 2022 at 7:13 PM

The needed yaml files in order to create a namespace inside of an empty k8s cluster. Useful if you want to associate everything in one place. Also shows how to add all values to the custom namespace without having to set namespace:

./k8s/namespace.yaml

apiVersion: v1
kind: Namespace
metadata:
  name: namespace-name

./k8s/kustomization.yaml

namespace: namespace-name
resources:
  - ./namespace.yaml

And to apply this to your specific context

kubectl apply -k ./k8s
