How to Deploy this Website

This assumes you have built the assets via yarn create::static already

Step 0: Make changes and commit to git main

git add . && git commit -m "My Message"

Step 1: Update Origin

git push

Step 2: SSH into DO Droplet

ssh username@ip

Step 3: Go to directory

cd my/directory/place

Step 4: Pull Origin

git pull

Step 6 (optional): Restart Docker-Compose

docker-compose up --build -d

Note: This is only needed when you need to restart or otherwise modify the running NGINX

Notes:

  • This is being done by the Github Actions defined in the repository. You should only have to build and push to origin and the system should take care of the rest.
